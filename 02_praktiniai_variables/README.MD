# Užduočių patobulinimai
Kai kurios užduotys buvo šiek tiek patobulintos (turėjau laisvo laiko 🙂).

## 02_04.DifferentTypesOfInput
Tikrinama, ar vartotojas tikrai įvedė tinkamo tipo reikšmes: `integer`, `double`, `boolean`.

## 02_05.SecondsInADay
Tikrinama, ar įvestas sveikasis skaičius. Taip pat diena negali būti `0` arba neigiama.

## 02_06.SumOfThreeNumbers
Yra galimybė naudoti daugiau nei 3 skaičius aritmetikai.

```java
String[] steps = { "first", "second", "third", "...", "n-th" };
```
Įvedant skaičių, jei įvedamas ne sveikasis skaičius, jis yra ignoruojamas, o aritmetika atliekama toliau.

## 02_07.MultiplicationFormula
Taip pat kaip ir **02_06.SumOfThreeNumbers**, yra galimybė dauginti daugiau nei 2 skaičius.
Nesveikieji skaičiai yra ignoruojami, tačiau aritmetika atliekama toliau.

## 02_08.AverageOfThreeNumbers
Taip pat kaip ir **02_06.SumOfThreeNumbers**, yra galimybė įvesti daugiau nei 3 skaičius, priklausomai nuo steps masyvo:
```java
String[] steps = { "first", "second", "third", "..." , "n-th" };
```
Įvedami tik sveiko tipo skaičiai. Jei įvedamas netinkamas skaičius, jis yra ignoruojamas.
